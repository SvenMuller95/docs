### جائزہ

سب گراف اسٹوڈیو پر بلنگ کیسے کام کرتی ہے اس کے فوری ڈیمو کے لیے، ذیل میں ویڈیو کو چیک کریں:

<VideoEmbed youtube="UrfIpm-Vlgs" />

انوائسز ایک گاہک کی واجب الادا ادائیگی کی رقم کے بیانات ہیں اور عام طور پر سسٹم میں ہفتہ وار بنیادوں پر تیار کیے جاتے ہیں۔ آپ کو اپنی API کلیدوں کے استعمال سے پیدا کردہ کیوری کی فیس کی بنیاد پر فیس ادا کرنے کی ضرورت ہوگی۔ جبکہ گراف پروٹوکول ایتھریم مینیٹ پر کام کرتا ہے، بلنگ کا معاہدہ [Polygon](https://polygon.technology/) نیٹ ورک پر رہتا ہے تاکہ ٹرانزیکشن کے اوقات اور لاگت کو کم کیا جا سکے. یہ آپ کو اجازت دے گا:

- اپنے والیٹ سے GRT شامل کریں یا ہٹا دیں
- آپ نے اپنے اکاؤنٹ میں کتنی GRT شامل کی ہے، کتنی رقم آپ نے ہٹا دی ہے، اور اپنے انوائسز کی بنیاد پر اپنے بیلنس کا ٹریک رکھیں
- خود بخود انوائسز کی ادائیگی کے لیے تیار کردہ استفسار کی فیس کی بنیاد پر، جب تک آپ کے اکاؤنٹ میں کافی بلنگ بیلنس موجود ہے.

آپ کی انوائسز 4 حالتوں میں ہو سکتی ہے:

1. زیر التواء - انوائس قطار میں ہے جس کی ادائیگی آپ کے بلنگ بیلنس سے کی جائے گی
2. ادا کیا گیا - آپ کی انوائسزکامیابی کے ساتھ ادا کر دی گئی ہے
3. بلا معاوضہ - آپ کے بلنگ بیلنس میں ناکافی GRT ہیں
4. جزوی طور پر ادا کیا - انوائس کا صرف ایک حصہ ادا کیا گیا، کیونکہ کافی بلنگ بیلنس نہیں تھا

### اکاؤنٹ میں GRT شامل کرنا

گراف نیٹ ورک پر سوالات کرنے سے پہلے، آپ کو اپنے اکاؤنٹ میں GRT شامل کرنے کی ضرورت ہوگی۔ نیٹ ورک ٹریفک کے لحاظ سے اس میں تقریباً 10-30 منٹ لگیں گے۔ اپنے اکاؤنٹ میں GRT شامل کرنے کے لیے، درج ذیل اقدامات کریں:

1. اپنی پسند کے تبادلے یا ڈیسینٹرالائزڈ تبادلہ پر GRT اور ETH خریدیں
2. GRT اور ETH اپنے والیٹ میں بھیجیں
3. سب گراف اسٹوڈیو UI کا استعمال کرتے ہوئے GRT کو پولیگون سے بریج کریں

   a) پولیگون بریج پرGRT بھیجنے کے چند منٹوں کے اندر آپ کو 0.001 میٹک مل جائے گا۔ آپ [Polygonscan](https://polygonscan.com/) پر سرچ بار میں اپنے والیٹ کا پتہ ڈال کر ٹرانزیکشن کی نگرانی کر سکتے ہیں.

4. سب گراف اسٹوڈیو UI میں پولیگون پر بلنگ کنٹریکٹ میں برجڈ GRT شامل کریں۔ بلنگ کنٹریکٹ کا پتہ یہ ہے: [0x10829DB618E6F520Fa3A01c75bC6dDf8722fA9fE](https://polygonscan.com/address/0x10829DB618E6F520Fa3A01c75bC6dDf8722fA9fE).

   a) مرحلہ نمبر 4 مکمل کرنے کے لیے، آپ کو ایتھریم سے پولیگون نیٹ ورک پر سوئچ کرنا ہو گا. آپ سب گراف اسٹوڈیو کا استعمال کرتے ہوئے یہ کر سکتے ہیں.

اگر آپ نے کسی بھی ہفتے کے دوران اپنی API کلید استعمال کی ہیں، آپ کو اس مدت کے دوران آپ کی طرف سے پیدا کردہ استفسار کی فیس کی بنیاد پر ایک انوائس موصول ہوگی۔ یہ انوائس آپ کے بیلنس میں دستیاب GRT کے ذریعے ادا کی جائے گی۔ استفسار کے حجم کا اندازہ آپ کی ملکیت والی API کلیدوں سے کیا جاتا ہے۔ فیس واپس لینے کے بعد آپ کا بیلنس اپ ڈیٹ ہو جائے گا.

### سب گراف اسٹوڈیو میں انوائس کو سمجھنا

گراف نیٹ ورک استعمال کرتے وقت، ڈی ایپس اور صارفین GRT میں استفسار کی فیس ادا کرتے ہیں، پروٹوکول کا مقامی ورک یوٹیلیٹی ٹوکن۔ اکاؤنٹ کے لیے مثبت توازن برقرار رکھنا صارف یا ڈی ایپ کی ذمہ داری ہے.

استفسار کی فیس بیلنس کے بارے میں مطلع کرنے اور/یا چیک کرنے کے طریقے یہ ہیں:

- آپ بلنگ کی اطلاع ترتیب دے سکتے ہیں جو [یہاں](https://thegraph.com/studio/settings/) ای میل پتے پر بھیجی جائے گی.

- آپ بلنگ سب گراف API سے براہ راست [یہاں](https://thegraph.com/hosted-service/subgraph/graphprotocol/billing) استفسار کر سکتے ہیں

اس کے علاوہ، آپ [یہ ویڈیو](https://www.loom.com/share/a81de6ef11d64c62872ea210c58c6af5) دیکھ کر اپنے استفسار کی فیس بیلنس کو (دوبارہ) بھرنے کا طریقہ سیکھ سکتے ہیں.

> اہم: اگر دو ہفتوں سے زائد عرصے سے غیر ادا شدہ انوائسز ہیں، انڈیکسرز اکاؤنٹ سے وابستہ تمام API کلیدوں کے لیے سوالات پیش کرنا بند کر دیں گے۔ بلنگ بیلنس کی مکمل ادائیگی اور مستقبل کے استفسار کی فیس کے لیے اکاؤنٹ میں GRT شامل ہونے کے بعد انڈیکسرز خود بخود دوبارہ سوالات پیش کرنا شروع کر دیں گے.

آخر میں، [یہ ویڈیو](https://www.loom.com/share/b5fc533e48584cb694017392c80c75e0) دیکھ کر اپنے سوالات کے لیے قیمت، رفتار، ڈیٹا کی تازہ کاری (دوسرے عوامل کے ساتھ) کو ترجیح دینے کا طریقہ سیکھیں۔ اگر آپ کے کوئی سوالات ہیں، تو براہ کرم [The Graph Discord](https://discord.gg/fjDWCTZE) میں پوچھیں اور کمیونٹی کو مدد کرنے میں خوشی ہوگی.

### ملٹیسگ والیٹ سے اپنا اکاؤنٹ کیسے پُر کریں

ملٹیسگس سمارٹ کنٹریکٹس ہیں جو صرف اس نیٹ ورک پر موجود ہو سکتے ہیں جو وہ بنائے گئے ہیں، لہذا اگر آپ نے ایتھریم مینیٹ پر ایک تخلیق کیا ہے - یہ صرف مینیٹ پر موجود ہوگا۔ چونکہ ہماری بلنگ پولیگون کا استعمال کرتی ہے، اگر آپ GRT کوپولیگون پر ملٹیسیگ ایڈریس پر پلٹتے ہیں تو فنڈز ضائع ہو جائیں گے.

اس مسئلے پر قابو پانے کے لیے، ہم نے [ایک وقف شدہ ٹول](https://multisig-billing.thegraph.com/) بنایا ہے جو آپ کو ہمارے بلنگ کنٹریکٹ (ملٹی سیگ کی جانب سے) پر ایک معیاری والیٹ / EOA (ایک نجی کلید کے ذریعے کنٹرول کردہ اکاؤنٹ) کے ساتھ GRT جمع کرنے میں مدد کرے گا.

آپ ہمارے ملٹی سگ بلنگ ٹول تک یہاں رسائی حاصل کر سکتے ہیں: https://multisig-billing.thegraph.com/

یہ ٹول درج ذیل مراحل سے گزرنے میں آپ کی رہنمائی کرے گا:

1. اپنے معیاری والیٹ / EOA کو جوڑیں (اس پرس میں کچھ ETH کے ساتھ ساتھ GRT کا بھی ہونا ضروری ہے جسے آپ جمع کرنا چاہتے ہیں)
2. GRT سے پولیگون بریج کریں۔ ٹرانزیکشن مکمل ہونے کے بعد پل کی منتقلی کو حتمی شکل دینے کے لیے آپ کو 7-8 منٹ انتظار کرنا پڑے گا.
3. ایک بار جب آپ کا GRT آپ کے پولیگون بیلنس پر دستیاب ہو جاتا ہے تو آپ اسے بلنگ کنٹریکٹ میں جمع کر سکتے ہیں اور اس ملٹی سگ ایڈریس کی وضاحت کرتے ہوئے جو آپ `Multisig Address` فیلڈ میں فنڈنگ ​​کر رہے ہیں.

ایک بار ڈپازٹ ٹرانزیکشن کی تصدیق ہو جانے کے بعد آپ [سب گراف اسٹوڈیو](https://thegraph.com/studio/) پر واپس جا سکتے ہیں اور API کیز بنانے کے لیے اپنے Gnosis Safe Multisig کے ساتھ جڑ سکتے ہیں اور ان کا استعمال کر سکتے ہیں۔ سوالات پیدا کریں.

وہ سوالات انوائسز تیار کریں گے جو ملٹی سیگ کے بلنگ بیلنس کا استعمال کرتے ہوئے خود بخود ادا کیے جائیں گے.
